# Minimal Stopwatch

**A distraction-free, high-precision multi-stopwatch for deep work.**

Minimal Stopwatch is a clean, keyboard-first stopwatch application designed for developers, designers, and anyone who needs to track time across multiple tasks without clutter. It features a focused "Zen Mode" to eliminate distractions and precise drift-free timing logic.

- **Multiple Timers**: Track several tasks simultaneously. Name them, reorder them, and manage them independently.

## ðŸš€ Getting Started

### Using the App
1. **Create a Timer**: Click the **+** button in the bottom right or press `N`.
2. **Start/Stop**: Click the timer digits or press `Space` to toggle the selected timer.
3. **Focus**: Click the scan icon (or press `F`) to isolate a timer.
4. **Zen Mode**: Shift+Click the scan icon (or press `Z`) to go full-screen.
5. **Reorder**: Drag timers using the handle on the left.

### Keyboard Shortcuts

| Key | Action |
| :--- | :--- |
| **Space** | Start / Stop active timer |
| **N** | Create new timer |
| **Arrow Up** | Select previous timer |
| **Arrow Down** | Select next timer |
| **F** | Toggle **Focus Mode** (Browser window) |
| **Z** | Toggle **Zen Mode** (Fullscreen) |
| **Esc** | Exit Focus or Zen mode |
